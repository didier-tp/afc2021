inscrire abonné

UMLUseCase
bibliotheque :: ModelUseCase :: inscrire abonné

Description

scénario nominal:

  1. vérifier si nouvelle inscription ou pas
  2. le responsable abonnement renseigne infos sur abonné (nom, prénom, adresse, ...)
  3. percevoir paiement si abonnement payant
  4. le responsable abonnement déclenche enregistrement des infos sur l'abonné et générer nouveau numéro abonné.
  5. imprimer carte d'abonné et afficher résumé sur opérations effectuées
